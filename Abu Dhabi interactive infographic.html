Regression models are used to create predictive models with quantitative data that link an independent variable with several dependent variables that affect it.
Clustering models are created from unlabelled datasets. These tools attempt to find groups of data points to cluster similarly and then fit new data into one of these clusters.
Classification models are created from labelled datasets. These tools attempt to place new data into the categories defined when creating the model with the labelled data.
